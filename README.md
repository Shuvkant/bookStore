# Nodejs FES Template

# Pre-requisites

- Install [Node.js](https://nodejs.org/en/) version 8.0.0

# Getting started

- Clone the repository

```zsh
  git clone https://github.com/Shuvkant/bookStore.git
```

## Move to Bookstore directory

```
cd boookStore
```

## Starting fronted

```
npm run dev
```

## Starting Backend

```
npm start
```

## Running on

```
http://localhost:5173/
```
